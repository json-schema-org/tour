# Comment 

To add comments in  your JSON Schema, use the `$comment` [keyword](https://json-schema.org/learn/glossary#subschema). The `$comment` keyword is a **string** that provides additional information about the [schema](https://json-schema.org/learn/glossary#schema). 

```json highlightLineStart={3}
{
  "type": "string",
  "$comment": "This is a comment"
}
```

# Default

You can set a default value for a property using the `default` keyword. The `default` keyword is used to provide a default value for a property. 

```json highlightLineStart={3}
{
  "type": "string",
  "default": "Hello World"
}
```

> **Remember:** These keywords are not used for validation purposes. They are used to provide additional information about the schema.
